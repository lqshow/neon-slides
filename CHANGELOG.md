# Changelog

## [0.1.0] - 2026-04-14

首个开源版本。

### 组件
- `.cover` — 首页封面屏(居中构图,84px 大标题,橙色发光分隔线)
- `.hero` — 章节过渡屏(超大描边编号 + 右侧标题,带进场动画)
- `.split` — 半屏图文(支持 `.reverse` 左右翻转,橙/蓝光晕边框)
- `.steps` — STEP 流程卡(橙蓝交替)
- `.compare` — 左右对比卡
- `.layers` — 三栏 Layer 架构卡(支持 icon box 和缩略图两种模式)
- `.list-grid` — 2 列清单网格
- `.code` — 代码块(橙色左边框 + 三色语法高亮)

### 导航
- 键盘 `←` `→` `Space` 翻页
- `Home` / `End` 跳首末页
- `G` 呼出跳转面板,输入页码回车直达
- ≤15 页用圆点导航(可点击),>15 页自动切换为进度条
- 所有圆点和进度条都支持点击跳转

### 设计 token
- 橙 `#F97316` / 蓝 `#3B82F6` 二元主色
- 字体 Noto Sans SC + Space Grotesk
- 深色底 `#0A0A0A` + 角落径向光晕

### 文档
- `SKILL.md` — 触发规则、组件清单、屏位规则
- `components.md` — 9 类组件的完整用法示例和反例
- `template.html` — 4 屏真实示例(封面 + 概念 + 架构 + 过渡)
